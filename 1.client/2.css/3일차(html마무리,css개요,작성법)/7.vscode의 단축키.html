vscode에서 자주 사용되는 단축키

  1. html 문서 템플릿 작성법 -> html 파일작성 후, "! enter " or "html:5 enter"
  2. 태그 작성하는 방법 => 태그이름 enter => 자동완성기능 ex) div enter
  3. 같은 태그를 여러개 생성하는 방법 -> 태그명*n(반복할 수) ex) div * 5
  4. 태그 중첩해서 생성하는 방법 => 태그명>태그명2>태그명3 ex) div>ul>ol*3
        1. 인접한 태그 생성하는 방법 ex) div+h1+h2+h3+span

        2. 아이디 선택자 설정 => 태그명#아이디명{내용추가} ex) div#name{내용추가} 생성
          내용추가
        3. 클래스 선택자 설정 => 태그명.클래스명 ex) div.test
        4. 태그{입력하고 싶은 텍스트} 입력후 enter or Tab ex) p{안녕하세요}

          안녕하세요

        5. 숫자삽입(숫자 1씩증가) ex) div#name$n (아이디,클래스,내용추가 선택할 때)
          ex2) div{텍스트}.test$*3
          텍스트
          텍스트
          텍스트
        6. ^입력시 상위계층으로 이동(중첩된 상황에서 상위게층으로 이동할 때, 여러번 사용가능) => 자식태그로 만들어줌 (편입하는것) ex) div>ul>li^p

        7. ^^p 한단게 아래로 이동 형식 ex) div>ul>li^^p

        8. (태그,,,)그룹으로 묶어주는 경우 ex) div>(ul>li)+ol>li 입력 후 enter 친 경우 => 그룹으로 묶어줄 경우에 맨 마지막 괄호는 생략하기
        9. 자동완성기능 div#myid.classA.classB 입력 후 enter div#myid.classA (id와 클래스 선택자를 동시에 생성)
